Marion Burk Knott Hall Website

I was given the position of Webmaster of my dorm, Marion Burk Knott Hall, in April of 2017. The dorm had websites already up, but the dorm presidents wanted the official website to have the ".nd.edu" to legitimaize it. After doing research on how to obtain such a domain for our dorm, I learned that there were limited options as Notre Dame wanted to ensure that websites under there domain lived up to University Standards. For this reason, I created a new website using WordPress on Notre Dame's sites.nd.edu network. Residents were impressed with the site and it is now the second search on Google for "Knott Hall".

This Website!

I made this website using HTML and CSS. It is hosted on github pages. I used Bootstrap to incorporate some slick features. Making this website helped me practice HTML and CSS, which I learned in high school in my Web Technologies class but I hadn't really applied them ever since. This forced me to do some more research in web development to figure out how to make this site look exactly the way I wanted it. Overall, I enjoyed the experience as it helped me improve my web development skills and allowed me to express my creativity.
